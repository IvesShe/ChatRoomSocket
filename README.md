# Socket聊天室(node.js + socket.)

  ## Server端相關安裝
  ```
  npm init -y
  npm install express@4.15.2
  npm i socket.io --save
  npm install nodemon -g  
  npm install --save-dev nodemon 
  nodemon
  ```

  ## 瀏覽器開啟客戶端(至少開2組以上比對使用)
  http://localhost:3000
